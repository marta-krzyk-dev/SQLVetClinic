# SQLVetClinic
An example database application for a vet clinic made with:
- MS Access,
- SQL,
- PL/SQL,
- Visual Basic.

## Features:
- individual forms for pets, vets, owners, vet specializations, animal species. 
- forms for adding, editing, deleting records,
- reports.
